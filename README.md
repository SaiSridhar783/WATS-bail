_Adapted to Multi-device_

## Clone this project

```bash
> git clone https://github.com/SaiSridhar783/WATS-bail.git
```

## Install the dependencies:

Before running the below command, make sure you're in the project directory that
you've just cloned!!

```bash
> cd WATS-bail
> npm i
```

### Usage

```bash
> npm start
> Scan QR code
```
